# Render function

Render function in Vue 2:

render(h) {
  return h('div', {
    attrs: {
      id: 'foo'
    },
    on: {
      click: this.onClick
    }
  }, 'hello');
}

Render function in Vue 3:

  • Flat props structure
  • Globally imported h helper
import { h } from 'vue';

render() {
  return h('div', {
    id: 'foo',
    onClick: this.onClick,
  }, 'hello');
}
